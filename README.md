# GraDeMarCo-WPF

このプログラムは、セラミックスや金属組織のSEM画像から粒径を求めるための画像処理を行います。

## 画像処理の流れ
1. SEM画像の使用する範囲を定めます。
2. Planimetric法で使用する円を描画します。
3. 画像について平滑化、エッジ検出、二値化を行い、粒子を自動で検出します。
4. 自動検出した粒子を手動で訂正および追加します。
5. 検出した粒子を数えます。

## Planimetric法について
1. 組織のSEM画像にある半径の円を書きます。
2. 円内の粒子および円周上の粒子数を数えます。
3. 円の半径および粒子の数から、平均粒径を求めます。
